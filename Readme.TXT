The folder contains the Final Project in VLSI-testing class at NTU in 2019-2 semester,
with the following files:
-	Introduction File: doc/Final_project, doc/Goel_PODEM_TC81
-	Report       File: Project_PPT, Proejct_report
-	Coding     Folder: /src
-	database   Folder: /sample_circuits, /tdf_patterns

One program and one report in this project, and the description of project is in
"doc/Final_project".
This project is to provide (N=1~8) N-detected Automatic-Test-Pattern-Generation function.
That is, to generate test pattern for detecting faults on the circuit as TESTING.
